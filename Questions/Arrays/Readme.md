Question
--
Given N array elements, count number of elements having atlist 1 element, greater than itself. (No library functions)

TC: O(N)

SC: O(1)

Example 1: 
--
arr[7] = [-3, -2, 6, 8, 4, 8, 5]

Output: 5

Example 2: 
--
arr[7] = [-3, -2, 6, 8, 4, 8, 5]

Output: 5

Example 3:
--
arr[10] = [2, 5, 1, 4, 8, 0, 8, 13, 8, 9]

Output: 9
