Question
--
Given N array elements, check if there exists pair [i,j] such that arr[i] + arr[j] == k & i!=j

Note: i & j are index values, k is the given sum

TC: O(N<sup>2</sup>)

SC: O(1)

Example 1: 
--
arr = [3, -2, 1, 4, 3, 6, 3]

k=10

Output: true

Example 2: 
--
arr = [2, 4, -3, 7]

k=5

Output: false

Example 3:
--
arr = [2, 4, -3, 7]

k=8

Output: false
